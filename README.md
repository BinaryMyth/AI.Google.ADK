# AI.Google.ADK
Learning Google's Agent Development Kit

This repository is based on below Kaggle course:

https://www.kaggle.com/learn-guide/5-day-agents

Setup instructions:

1. pip install google-adk
2. Get API Key from: https://aistudio.google.com/app/api-keys
3. Set API Key in env variable "GOOGLE_API_KEY"
4. Run any of the Python files in this repository
